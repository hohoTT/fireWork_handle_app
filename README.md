# fireWork_handle_app

用于方便解决消防过程中计算水量以及泡沫量的应用工具

在日常的消防过程中，水量以及泡沫量是根据现场的火情来计算出来的。

这里我们考虑的主要分为以下两种情况：
    1. 一般火灾
    2. 油罐火灾
    
计算的是上述两种情况下的的火灾剂用量和供水压力的计算
